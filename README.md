IT Portal API, kurum içi IT destek süreçlerini yönetmek amacıyla geliştirilmiş bir backend servisidir. Sistem, kullanıcıların IT talepleri oluşturmasını, bu taleplerin IT personeli tarafından yönetilmesini ve tüm süreçlerin merkezi bir platform üzerinden takip edilmesini sağlar.

Proje aşağıdaki teknolojiler kullanılarak geliştirilmiştir:

ASP.NET Core Web API
Entity Framework Core
JWT Authentication
Role Based Authorization
Repository & Service Pattern
MySQL / MSSQL / SQLite

Amaç, IT süreçlerini daha düzenli, izlenebilir ve yönetilebilir hale getiren bir altyapı oluşturmaktır.
